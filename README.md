# Joe | Software Engineer Portfolio

A modern, responsive personal portfolio website designed to showcase software engineering projects, technical skills, and professional experience.

🚀 **Live Demo:** [https://joe-h.netlify.app](https://joe-h.netlify.app)

## ✨ Features

*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices using CSS Grid and Flexbox.
*   **Modern UI/UX:** Dark mode aesthetic with neon accents, smooth scrolling, and subtle animations.
*   **Performance:** Optimized assets with lazy loading and asynchronous decoding for fast load times.
*   **Contact Form:** Integrated with Netlify Forms for serverless submission handling and spam protection (Honeypot).
*   **SEO Friendly:** Semantic HTML5 structure with proper meta tags and Open Graph support.

## 🛠️ Tech Stack

*   **HTML5:** Semantic markup.
*   **CSS3:** Custom properties (variables), media queries, and keyframe animations.
*   **Netlify:** Hosting and form backend services.

## 📂 Project Structure

```text
├── assets/          # Images and icons (favicon, profile)
├── index.html       # Main HTML structure
├── style.css        # Global styles and animations
└── README.md        # Project documentation
```

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/justjoe19/Personal-Website.git
    ```
2.  **Open the file:**
    Simply open `index.html` in your preferred web browser. No build step or server is required for the static content.
